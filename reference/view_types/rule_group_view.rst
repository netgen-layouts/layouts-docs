``rule_group_view``
===================

.. note::

    Available only in Enterprise Edition

This view is used to render ``Netgen\Layouts\API\Values\LayoutResolver\RuleGroup``
entity.

Available variables
-------------------

+---------------+----------------------------------------+
| Variable name | Description                            |
+===============+========================================+
| ``group``     | The rule group which is being rendered |
+---------------+----------------------------------------+
