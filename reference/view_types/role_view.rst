``role_view``
=============

.. note::

    Available only in Enterprise Edition

This view is used to render ``Netgen\Layouts\Enterprise\API\Values\Role\Role``
entity.

Available variables
-------------------

+---------------+-----------------------------------+
| Variable name | Description                       |
+===============+===================================+
| ``role``      | The role which is being rendered  |
+---------------+-----------------------------------+
