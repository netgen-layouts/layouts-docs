``role_policy_view``
====================

.. note::

    Available only in Enterprise Edition

This view is used to render ``Netgen\Layouts\Enterprise\API\Values\Role\Policy``
entity.

Available variables
-------------------

+----------------+-----------------------------------------+
| Variable name  | Description                             |
+================+=========================================+
| ``policy``     | The role policy which is being rendered |
+----------------+-----------------------------------------+
